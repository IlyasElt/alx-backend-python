# 0x03. Unittests and Integration Tests

This module teaches how to write **unit tests** and **integration tests** in Python using `unittest` and `parameterized`.  
It includes:

- `utils.py` – helper functions including `access_nested_map`, `get_json`, and `memoize`.
- `test_utils.py` – unit tests for `access_nested_map`.

## Usage

Run tests with:

```bash
python -m unittest test_utils.py
